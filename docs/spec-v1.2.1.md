# v1.2.1 仕様書 — タブ名改善・会計コンパクト化・アプデ履歴・店舗切替

## 概要

計算画面のスピード感・効率性を最優先に改善する。  
会計タブは**縦スクロール不要の1画面表示**を実現。  
加えてUI名称改善、アップデート履歴表示、店舗切替の利便性向上を行う。

---

## 1. タブ名変更

| 変更前 | 変更後 | 理由 |
|--------|--------|------|
| 単価もの | **オーダー** | 直感的で分かりやすい |
| AI | **予算** | 機能内容を正確に表す |

基本情報・会計はそのまま維持。

---

## 2. 予算タブの説明文追加

予算タブを開いた際、入力エリアの上に簡潔な説明を表示：

```
予算に対しての最適な注文内容をAIが自動計算します。
予算と退店時間を設定してください。
```

- 1行目: 太字
- 2行目: gray-400 で補足

## 3. 会計タブ 1画面コンパクト化

### 3-1. 設計方針
- **スマホ1画面（約600px高）にすべて収まる**
- 縦スクロールは極力排除
- 最重要情報（合計金額）を大きく、補助情報はコンパクトに

### 3-2. レイアウト（上から下）

```
┌──────────────────────────────┐
│  現在時刻: 22:30  [編集]     │  ← 小さく右上に
│                              │
│     ¥ 45,000               │  ← 大きな合計金額（text-4xl）
│     現在のお会計 (税込)       │
│                              │
│  ┌ 1つ前: ¥32,000 ──── ▼ ┐ │  ← 1行で前セット表示（タップで展開）
│                              │
│  ── 内訳 ─────────────── ▼ │  ← 折りたたみ（デフォルト閉）
│                              │
│  ── 料金スケジュール ────── │  ← コンパクト横並び
│  21:00  ¥12,000             │
│  22:00  ¥32,000             │
│  23:00  ¥45,000  ← 現在     │
│  24:00  ¥58,000             │
│  25:00  ¥71,000             │
└──────────────────────────────┘
```

### 3-3. 具体的な変更点

#### 合計金額エリア
- 現行: `text-4xl` → 維持
- 時刻表示を合計の上に小さく移動（`text-xs`）
- 「現在のお会計 (税込)」は `text-xs` に縮小

#### 前セット料金
- 現行: 見出し + 金額 + 折りたたみ内訳
- 変更: **1行のインライン表示** `1つ前: ¥32,000 ▼`（タップで内訳展開）
- 該当なし時は非表示

#### 内訳セクション
- **デフォルト折りたたみ**（現行は常時表示）
- `内訳 ▼` をタップで展開
- 展開時もコンパクト（padding縮小、font-size縮小）

#### 料金スケジュール
- 現行: テーブル形式 + タップ展開内訳
- 変更: **コンパクトリスト形式**
  - 各行: `時間  金額` のみ（内訳はタップ展開維持）
  - 現在の時間帯をハイライト（`text-[var(--accent-color)]` + `◀`マーク）
  - テーブルヘッダー削除、行間縮小
  - 5行程度で収まるよう `text-sm` に

### 3-4. 会計時刻オーバーライド
- 現行: テキスト「現在時刻: XX:XX」タップ → input[type=time]
- 変更: タップ → **ButtonTimePicker** をインライン展開
- ButtonTimePickerが表示されている間は他のセクションを非表示にしてスペース確保

## 5. 店舗切替の管理者外アクセス

### 5-1. 変更内容
- **店舗切替**（選択のみ）はパスワードなしで利用可能
- **店舗追加・削除・名前変更** は管理者ログイン後のみ

### 5-2. UI配置

SettingsPage のレイアウトを変更:

```
┌──────────────────────────────────┐
│  ◉ 設定                          │
│                                   │
│  ── 店舗切替 ──                   │  ← パスワード不要
│  [GENTLY DIVA]  [店舗B]  [店舗C] │  ← ボタンで切替
│                                   │
│  ── 管理者モード ──               │
│  [パスワード入力] [解除]          │  ← 従来通り
│                                   │
│  ── 過去のアップデート ──          │  ← パスワード不要
│  v1.2.1 ...                       │
│  v1.2.0 ...                       │
└──────────────────────────────────┘
```

### 5-3. 店舗切替UI
- 横並びボタン（2-3個なら均等幅、4個以上はスクロール）
- アクティブ店舗: `bg-[var(--gold-color)] text-black`
- 非アクティブ: `border-[var(--border-color)] bg-[var(--input-bg)]`
- タップで即座に切替（確認ダイアログなし）
