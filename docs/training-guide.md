## 第0章 基礎知識

- **営業時間**: 20:00〜25:00（翌1:00）
- **TAX/SVC**: 合計 **35%**（消費税10% + サービス料25%）
- **端数処理**: 最終合計は **百円単位切り上げ**
- **例**: ¥12,345 → **¥12,400**

## 第1章 客種（お客様の種類）を見分ける

| 客種 | 来店回数 | 特徴 |
|------|----------|------|
| **新規** | 1回目 | 初めてのお客様 |
| **R（リターン）** | 2回目 | チケットの有無で料金が変わる |
| **正規** | 3回目以降 | 常連のお客様 |

### R の2種類

| 種類 | 条件 | 料金 |
|------|------|------|
| **Rチケット有り（チケット有り）** | 新規来店から **2週間以内** かつチケット所持 | お得な料金 |
| **Rチケット無し（チケット無し）** | 2週間超過 / チケット紛失・忘れ | 通常料金 |

### 判別フロー

1. 初めてのお客様？ → **新規**
2. 2回目の来店？
   - チケットあり＆2週間以内 → **Rチケット有り**
   - チケットなし or 2週間超過 → **Rチケット無し**
3. 3回目以降 → **正規**

### T.C（テーブルチャージ）について

- **Rチケット無し** と **正規** のみに発生（来店時に必ずかかる）
- 新規・Rチケット有りは T.C なし（¥0）

### サービスドリンク（Rチケット無し・正規のみ）

来店時に以下のいずれかを **無料** で提供（会計に含まない）:

- **ハーフボトル1本**（以下から選択）: 鏡月 / 神の河 / 一刻者 / 梅酒 / 杏露酒 / 林檎酒
- **ペットボトル2本**

※ 両方オーダーした場合は **お酒の方をサービス** で出す

### 覚えるポイント

- 新規・正規 → 1セット **1時間**
- R系 → 1セット **2時間**（チケット有無に関わらず）
- 新規の同伴は **無料（¥0）**
- 新規でオーダー無し → 非課税

## 第5章 割引イベント

### イベント別の割引効果

| イベント | 開催条件 | セット半額 | カン半額 | ショット半額 | シャンパン半額 |
|----------|----------|:---------:|:--------:|:-----------:|:-------------:|
| 女子会デー | 隔週木曜 / 2人以上来店 | × | ○ | ○ | ○（ブルー〜ゴールド） |
| 感謝デー | 月の第一営業日 | ○（正規のみ） | ○ | ○ | × |
| セブンラック | その日の来店7組目 | × | ○ | ○ | ○（ブルー〜ゴールド） |
| スタートシャンパン | 22時まで / 全卓で最初の1本 | × | × | × | ○（リステル〜ゴールド） |

### 各イベントの詳細

- **女子会デー**: 隔週木曜日。お客様が **2人以上** での来店が条件
- **感謝デー**: 毎月の **第一営業日**。セット料金+カン+ショットが半額
- **セブンラック**: その日の **来店7組目** のお客様が対象。セブンラックタワー（¥150,000 税込）が注文可能
- **スタートシャンパン（スタシャン）**: その日の **全卓の中で最初のシャンパン** がリステル〜ゴールドまで半額。**22時まで** が対象

### 併用ルール（重要）

- **女子会デー ⇔ 感謝デー** → 同時にできない（排他）
- **セブンラック** → 女子会・感謝デー の **どちらとも併用OK**
- 感謝デー中 → セット半額が **自動でON**（手動で切替不可）
- **セット半額は正規のみ**（新規・Rには効かない）
- **二重半額は禁止**（すでに半額のものをさらに半額にはしない）

### 半額の計算方法

| 種類 | 計算 |
|------|------|
| セット・延長の半額 | 切り捨て（例: ¥3,000 → ¥1,500） |
| カンの半額 | 特殊価格 **¥700固定**（¥1,500の半額¥750ではない） |
| ショット系の半額 | ¥3,000 → ¥1,500 |

## 第8章 計算の流れ

### 5ステップで計算

1. **客種と入店時刻を確認** → セット料金・延長料金を決定
2. **ゴールドカードがあれば上書き**（セット¥0、延長¥1,000）
3. **滞在時間から延長本数を計算**
   - 滞在時間（切り上げ） − 1セットの時間 = 延長本数
   - 例: 2時間10分 → 3時間（切り上げ）、1セット1時間なら延長2本
4. **小計を出す**
   - セット + 延長 + 指名料 + 同伴料 + T.C + 複数指名 + オーダー合計
5. **TAX/SVC を加算して百円切り上げ**
   - 小計 × 1.35 → 百円単位切り上げ = **最終合計**

## 第9章 シナリオ別計算例

### 例1: 新規 × オーダー無し（非課税）

**前提**
- 客種: 新規（セット¥1,000）
- 入店: 20:30 → 会計: 22:00
- オーダー: なし

**計算**
1. 滞在: 1時間30分 → 切り上げ **2時間**
2. 1セット1時間 → 延長 **1本**
3. セット ¥1,000 + 延長 ¥2,000 + 指名 ¥1,000 = ¥4,000
4. オーダー無しの新規 → TAX/SVC **0%**
5. **合計: ¥4,000**

---

### 例2: Rチケット有り × カン2本

**前提**
- 客種: Rチケット有り（チケット有り）
- 入店: 20:00 → 会計: 23:00
- オーダー: カン × 2（¥1,500 × 2）

**計算**
1. 滞在: 3時間
2. 1セット2時間 → 延長 **1本**
3. セット ¥0 + 延長 ¥1,000 + 指名 ¥1,000 + カン ¥3,000 = ¥5,000
4. TAX/SVC: ¥5,000 × 35% = ¥1,750
5. 合計: ¥6,750 → 切り上げ **¥6,800**

---

### 例3: 正規（21:00前入店）× 同伴 + 複数指名 + シャンパン

**前提**
- 客種: 正規（20:30入店）
- 会計: 24:00（3時間30分滞在）
- 同伴あり、追加指名2人
- オーダー: SPLブルー × 1（¥35,000）

**計算**
1. 滞在: 3時間30分 → 切り上げ **4時間**
2. 1セット1時間 → 延長 **3本**
3. セット ¥2,000 + 延長 ¥9,000（¥3,000×3）= ¥11,000
4. 指名 ¥3,000 + 同伴 ¥3,000 + T.C ¥500 + 追加指名 ¥6,000（¥3,000×2）= ¥12,500
5. オーダー: ¥35,000（正規・イベント無しなのでシャンパン半額なし）
6. 小計: ¥11,000 + ¥12,500 + ¥35,000 = ¥58,500
7. TAX/SVC: ¥58,500 × 35% = ¥20,475
8. 合計: ¥78,975 → 切り上げ **¥79,000**

**前提**

### 例5: 正規 × ゴールドカード

**前提**

## 付録A 用語集

| 用語 | 意味 |
|------|------|
| **セット料金** | 席についた時にかかる基本料金 |
| **延長** | 1セットの時間を超えた分、1時間ごとに追加される料金 |
| **指名料** | キャストを指名するときの料金 |
| **同伴** | お客様が店外からキャストと一緒に来店すること。追加料金が発生 |
| **T.C** | テーブルチャージ。席に対する追加料金 |
| **TAX/SVC** | 税金＋サービス料（合計35%） |
| **カン** | 缶ドリンク |
| **ショット系** | ショットグラスで提供されるお酒 |
| **SPL〜** | シャンパンのブランド名 |
| **21:00前 / 21:00以降** | 入店時刻による料金区分（正規のみ） |
| **ゴールドカード** | セット・延長料金を大幅に割引する特別カード |
| **フリー回し飲み** | 新規のセット料金プラン（¥1,000 / ¥3,000 / ¥5,000） |
| **サービスドリンク** | Rチケット無し・正規に無料で提供されるドリンク |
| **スタシャン** | スタートシャンパン。その日の全卓最初のシャンパンが半額 |
| **半額** | 特定の条件で価格が半分になる割引 |
