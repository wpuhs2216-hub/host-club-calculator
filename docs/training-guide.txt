  Rチケット有り（チケット有り）

  Rチケット無し（チケット無し）

▼ 判別フロー
  初めて？         → 新規
  2回目？
    チケットあり＆2週間以内 → Rチケット有り
    それ以外               → Rチケット無し
  3回目以降？      → 正規

【7】計算の流れ（5ステップ）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ① 客種と入店時刻を確認 → セット・延長料金を決定
  ② ゴールドカードがあれば上書き
  ③ 滞在時間から延長本数を計算
     滞在時間（切り上げ） − 1セットの時間 = 延長本数
  ④ 小計を出す
     セット+延長+指名+同伴+T.C+複数指名+オーダー合計
  ⑤ TAX/SVC を加算して百円切り上げ
     小計 × 1.35 → 百円切り上げ = 最終合計

【8】計算例
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

── 例1: 新規 × オーダー無し ──
  セット¥1,000 / 20:30入店 / 22:00会計
  滞在1.5h → 切り上げ2h / 1セット1h → 延長1本
  ¥1,000 + ¥2,000 + 指名¥1,000 = ¥4,000
  オーダー無し新規 → TAX 0%
  ▶ 合計: ¥4,000

── 例2: Rチケット有り × カン2本 ──
  20:00入店 / 23:00会計
  滞在3h / 1セット2h → 延長1本
  ¥0 + ¥1,000 + 指名¥1,000 + カン¥3,000 = ¥5,000
  TAX 35%: ¥1,750
  ▶ 合計: ¥6,750 → 切上げ ¥6,800

── 例3: 正規（21:00前入店）× 同伴 + 複数指名2人 + SPLブルー ──
  20:30入店 / 24:00会計
  滞在3.5h → 切り上げ4h / 1セット1h → 延長3本
  ¥2,000 + ¥9,000 + 指名¥3,000 + 同伴¥3,000
    + T.C¥500 + 追加指名¥6,000 + SPLブルー¥35,000
  = ¥58,500
  TAX 35%: ¥20,475
  ▶ 合計: ¥78,975 → 切上げ ¥79,000

── 例4: 正規 × 女子会デー × SPLロゼ2本 ──
  21:30入店（21:00以降） / 24:30会計
  滞在3h / 1セット1h → 延長2本
  ¥5,000 + ¥6,000 + 指名¥3,000 + T.C¥500
    + SPLロゼ半額¥40,000×2 = ¥94,500
  TAX 35%: ¥33,075
  ▶ 合計: ¥127,575 → 切上げ ¥127,600

── 例5: 正規 × ゴールドカード ──
  20:00入店 / 23:00会計
  滞在3h / ゴールドで1セット2h → 延長1本
  ¥0 + ¥1,000 + 指名¥3,000 + T.C¥500 = ¥4,500
  TAX 35%: ¥1,575
  ▶ 合計: ¥6,075 → 切上げ ¥6,100
  ※ ゴールドカード中はセット半額は使えない

【10】用語集
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  セット料金 … 席についた時の基本料金
  延長       … 1セットの時間の超過分の追加料金（1時間ごと）
  指名料     … キャストを指名する料金
  同伴       … 店外からキャストと一緒に来店する料金
  T.C        … テーブルチャージ
  TAX/SVC    … 税金＋サービス料（合計35%）
  カン       … 缶ドリンク
  ショット   … ショットグラスのお酒
  SPL～      … シャンパンのブランド名
  21:00前/以降 … 入店時刻による料金区分（正規のみ）
  ゴールドカード … セット・延長を大幅割引する特別カード
  フリー回し飲み … 新規のセット料金プラン（¥1,000/¥3,000/¥5,000）
  サービスドリンク … Rチケット無し・正規に無料で提供されるドリンク
  スタシャン     … スタートシャンパン。全卓最初のシャンパンが半額
