■ 基礎知識
─────────────────────────────
・営業時間: 20:00～25:00（翌1:00）
・TAX/SVC: 合計 35%（消費税10% + サービス料25%）
・端数処理: 最終合計は百円単位切り上げ
  例: ¥12,345 → ¥12,400

【1】客種（お客様の種類）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  新規     … 1回目の来店
  R        … 2回目の来店
  正規     … 3回目以降の来店

▼ R の2種類
  R-within（チケット有り）
    → 新規来店から2週間以内 かつ チケット所持
  R-after（チケット無し）
    → 2週間超過 / チケット紛失・忘れ

▼ 判別フロー
  初めて？         → 新規
  2回目？
    チケットあり＆2週間以内 → R-within
    それ以外               → R-after
  3回目以降？      → 正規

▼ 併用ルール
  ・女子会デー ⇔ 感謝DAY → 同時にできない（排他）

  ・セット半額 → 正規のみ対象

▼ 半額の計算
  ・セット・延長の半額 → 切り捨て
  ・カンの半額 → ¥700固定（¥750ではない！）
  ・ショット系の半額 → ¥1,000

【7】計算の流れ（5ステップ）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ① 客種と入店時刻を確認 → セット・延長料金を決定
  ② ゴールドカードがあれば上書き
  ③ 滞在時間から延長本数を計算
     滞在時間（切り上げ） − 初回セット時間 = 延長本数
  ④ 小計を出す
     セット+延長+指名+同伴+T.C+複数指名+オーダー合計
  ⑤ TAX/SVC を加算して百円切り上げ
     小計 × 1.35 → 百円切り上げ = 最終合計

【8】計算例
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

── 例1: 新規 × オーダー無し ──
  セット¥1,000 / 20:30入店 / 22:00会計
  滞在1.5h → 切り上げ2h / 初回1h → 延長1本
  ¥1,000 + ¥2,000 + 指名¥1,000 = ¥4,000
  オーダー無し新規 → TAX 0%
  ▶ 合計: ¥4,000

── 例2: R-within × カン2本 ──
  20:00入店 / 23:00会計
  滞在3h / 初回2h → 延長1本
  ¥0 + ¥1,000 + 指名¥1,000 + カン¥3,000 = ¥5,000
  TAX 35%: ¥1,750
  ▶ 合計: ¥6,750 → 切上げ ¥6,800

── 例3: 正規（早番）× 同伴 + 複数指名2人 + SPLブルー ──
  20:30入店 / 24:00会計
  滞在3.5h → 切り上げ4h / 初回1h → 延長3本
  ¥2,000 + ¥9,000 + 指名¥3,000 + 同伴¥3,000
    + T.C¥500 + 追加指名¥6,000 + SPLブルー¥35,000
  = ¥58,500
  TAX 35%: ¥20,475
  ▶ 合計: ¥78,975 → 切上げ ¥79,000

── 例4: 正規 × 女子会デー × SPLロゼ2本 ──
  21:30入店（遅番） / 24:30会計
  滞在3h / 初回1h → 延長2本
  ¥5,000 + ¥6,000 + 指名¥3,000 + T.C¥500
    + SPLロゼ半額¥40,000×2 = ¥94,500
  TAX 35%: ¥33,075
  ▶ 合計: ¥127,575 → 切上げ ¥127,600

── 例5: 正規 × ゴールドカード ──
  20:00入店 / 23:00会計
  滞在3h / ゴールドで初回2h → 延長1本
  ¥0 + ¥1,000 + 指名¥3,000 + T.C¥500 = ¥4,500
  TAX 35%: ¥1,575
  ▶ 合計: ¥6,075 → 切上げ ¥6,100
  ※ ゴールドカード中はセット半額は使えない

【10】用語集
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  セット料金 … 席についた時の基本料金
  延長       … 初回セット超過分の追加料金（1時間ごと）
  指名料     … キャストを指名する料金
  同伴       … 店外からキャストと一緒に来店する料金
  T.C        … テーブルチャージ
  TAX/SVC    … 税金＋サービス料（合計35%）
  カン       … 缶ドリンク
  ショット   … ショットグラスのお酒
  SPL～      … シャンパンのブランド名
  早番/遅番  … 21:00入店を境にした料金区分（正規のみ）
  ゴールドカード … セット・延長を大幅割引する特別カード
