# GENTLY DIVA 会計計算機 操作ガイド

> アプリの使い方を画面ごとに解説します。
> 料金ルールの詳細は「会計研修ハンドブック」を参照してください。

---

## 画面の構成

アプリは大きく3つの画面に分かれています。

| 画面 | 内容 |
|------|------|
| **計算** | 伝票ごとの会計計算（メイン画面） |
| **全卓会計** | 全テーブルの伝票を一覧表示 |
| **設定** | 店舗・料金・メニューなどの設定 |

---

## 1. 伝票を作る

### 手順

1. サイドバーの「**+ 伝票追加**」をタップ
2. 以下を入力

| 項目 | 操作 |
|------|------|
| **客種** | 「新規」「R」「正規」のいずれかをタップ |
| **チケット** | R を選んだ場合 →「チケット有り」or「チケット無し」 |
| **セット料金** | 新規を選んだ場合 → ¥0 / ¥1,000 / ¥3,000 / ¥5,000 |
| **入店時間** | 時 → 十の位 → 一の位 の順にタップ |
| **同伴** | 該当する場合はチェックを入れる |

3. 「**作成**」をタップ → 伝票が開く

### ポイント

- 入店時間の入力を間違えたら「← 戻る」で前のステップに戻れる
- 同伴は後からでもON/OFFできる

---

## 2. 基本情報タブ

伝票を開くと、4つのタブが表示されます。最初のタブが「**基本情報**」です。

### 客種の変更

- 「**新規**」「**R**」「**正規**」ボタンで切り替え可能
- 新規 → セット料金の選択ボタンが出る
- R → チケット有り/無しの選択ボタンが出る

### 入店時間の変更

- 表示されている時刻をタップ → 時間ピッカーで再設定

### 同伴

- チェックで ON/OFF（+¥3,000）
- **新規のお客様には適用されない**

### 割引オプション（折りたたみ）

「割引オプション」をタップすると展開されます。

| 項目 | 操作 | 注意 |
|------|------|------|
| **セット料金半額** | トグルで ON/OFF | 感謝DAY中は自動ONで操作不可 |
| **女子会デー** | トグルで ON/OFF | 感謝DAYとは同時にできない |
| **お客様感謝DAY** | トグルで ON/OFF | 女子会デーとは同時にできない |
| **セブンラック** | トグルで ON/OFF | 他のイベントと併用OK |
| **ゴールドカード** | トグルで ON/OFF | セット半額とは併用不可 |
| **複数指名** | 「−」「+」で人数を調整 | 1人あたり ¥3,000 |

## 3. オーダータブ

ドリンクやシャンパンを追加する画面です。

### オーダーを追加する

1. 「**+ オーダーを追加**」をタップ
2. カテゴリを選ぶ（4つのタブ）

| タブ | 内容 |
|------|------|
| **定番** | ソフトドリンク、カクテル、レッドブル など |
| **シャンパン** | リステル〜SPLプラチナ（16種類） |
| **焼酎/果実酒** | ハーフボトル、JAPAN、黒霧島 など |
| **特殊/タワー** | オリシャン（自由入力）、テキーラスタンド など |

3. 商品をタップして選択（金色の枠がつく）
4. 半額対象の場合 →「**半額適用**」にチェック
5. 「**選択したアイテムを追加する**」をタップ

### オリシャン（自由入力）の追加

1. 「特殊/タワー」タブ →「オリシャン / その他」を選択
2. 商品名を入力（例:「オリジナルシャンパン」）
3. 金額を入力（例: 100000）
4. 「選択したアイテムを追加する」をタップ

### 数量の変更・削除

| 操作 | ボタン |
|------|--------|
| 1個ずつ増やす | **+** |
| 1個ずつ減らす | **−** |
| 5個ずつ増やす | **+5** |
| 5個ずつ減らす | **−5** |
| 削除 | **削除**（ピン留め商品は削除不可） |

### ピン留め商品（常時表示）

以下の3つは常に表示され、削除できません。

| 商品 | 価格 | 備考 |
|------|-----:|------|
| カン | ¥1,500 | 半額時 ¥700 |
| ペットボトル | ¥2,000 | |
| ショット系 | ¥2,000 | 半額時 ¥1,000 |

### 半額の注意点

- 新規・Rのシャンパン → **1本だけ**半額（「半額適用済み（1本制限）」と表示）
- 女子会デー・セブンラック → シャンパン**何本でも**半額
- カンの半額は ¥750 ではなく **¥700固定**

## 4. 会計タブ

計算結果を確認する画面です。

### 表示内容

| 項目 | 説明 |
|------|------|
| **現在のお会計（税込）** | TAX/SVC込みの最終合計（大きい数字） |
| **時刻表示** | 現在時刻ベースの計算。✎ をタップで時刻変更可能 |
| **1つ前** | 1セット前の会計金額（展開で内訳表示） |
| **内訳** | セット・延長・指名・オーダー・TAX/SVC の明細 |
| **料金スケジュール** | 時間帯ごとの合計金額一覧 |

### 時刻の変更

- 営業時間外の場合 →「営業時間外（1時間分）」と表示
- ✎（鉛筆アイコン）をタップ → 任意の時刻に変更可能

### 料金スケジュール

- 入店時刻から閉店（25:00）まで、1時間刻みで合計を表示
- 「お客様が○時まで滞在したらいくらか」が一目で分かる
- 各時間帯をタップすると内訳が展開される

## 5. 予算タブ（AIプランナー）

お客様の予算に合わせた注文を自動提案する機能です。

### 使い方

1. **目標金額**を入力（テキスト欄 or スライダー）
   - スライダー: ¥10,000〜¥200,000（¥5,000刻み）
2. **滞在時間**をタップして選択
   - 各時間帯のボタンが表示される
   - 金色 = 選択中 / 白 = 予算内 / 赤 = 予算オーバー
3. **おすすめ**が表示される
   - 「この時間内で追加できるもの」として候補が出る
   - 「**+ 追加**」をタップするとオーダーに反映

### 表示内容

| 表示 | 意味 |
|------|------|
| **残り** ¥XX,XXX | 予算に対してあといくら使えるか |
| **超過** ¥XX,XXX | 予算をいくらオーバーしているか |
| **あと少し予算を上げれば…** | もう少し予算があれば注文できる商品 |

## 6. 全卓会計

全テーブルの伝票を一覧で確認・操作する画面です。

### 表示の切り替え

| モード | 表示内容 |
|--------|----------|
| **現在** | 現時点での会計金額 |
| **ラストまで** | 閉店時の会計金額 |

### テーブルカード

各テーブルに以下の情報が表示されます。

- テーブル名 + 伝票数
- 各伝票の客種・入店時間
- 現在の金額 / ラストまでの金額

### 操作ボタン

| ボタン | 操作 |
|--------|------|
| **開く** | その伝票を計算画面で開く |
| **編集** | その場で客種・割引・オーダーを変更 |
| **移動** | 伝票を別のテーブルに移す |

### 編集モード

「編集」をタップすると、以下が表示されます。

- 客種の変更ボタン
- セット料金の変更（新規のみ）
- 入店時間の変更
- 各トグル（同伴・セット半額・イベント・ゴールドカード）
- オーダーの数量調整

### 伝票の移動

1. 「**移動**」をタップ
2. 移動先テーブルが表示される
3. 移動先をタップ → 完了

### 全卓リセット

- 画面上部の「**全卓リセット**」で全伝票を一括削除
- **注意: 元に戻せません**

## 8. よくある操作Q&A

**Q: 客種を間違えて作ってしまった**
→ 基本情報タブで客種ボタンをタップすれば変更できます

**Q: オーダーを間違えて追加した**
→ オーダータブで「削除」ボタンをタップ。ピン留め商品は数量を0にすればOK

**Q: 入店時間を間違えた**
→ 基本情報タブの時刻表示をタップして再入力

**Q: セブンラックタワーが見つからない**
→ 基本情報タブで「セブンラック」をONにしてください。ONにすると特殊/タワーに表示されます

**Q: シャンパンの半額が適用できない**
→ 新規・Rは1本制限です。「半額適用済み（1本制限）」と出たら、それ以上は半額にできません

**Q: 感謝DAY中にセット半額がOFFにできない**
→ 感謝DAYでは自動でONになり、手動切替できない仕様です

**Q: 予算タブで時間帯が赤い**
→ その時間帯まで滞在すると予算オーバーという意味です。前の時間帯を選ぶか予算を上げてください
