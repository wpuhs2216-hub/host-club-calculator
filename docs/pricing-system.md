# GENTLY DIVA 料金システム 完全版

> このドキュメントはホストクラブ会計計算機（PWA）に実装されている GENTLY DIVA の料金システムを網羅的に記載したものです。
> 設定値は `src/data/defaultStoreConfig.ts`、計算ロジックは `src/hooks/useCalculator.ts` を参照してください。
> 不明確な項目には `【要確認】` を付けています。必要があれば情報提供をお願いします。

---

## 1. 基本パラメータ

| 項目 | 値 |
|---|---|
| 店舗名 | GENTLY DIVA |
| 営業時間 | 20:00 〜 25:00（翌1:00） |
| 税 / サービス料（TAX/SVC）| 35%（一律） |
| 新規でオーダー無しの場合 | 非課税（0%） |
| LOキャップ時刻 | 24:38（翌 0:38） |

### 時刻の取扱い
- 入店時間・現在時刻は内部で「翌日時刻」を 24 以降として正規化（例: 翌 0:38 → 24:38）
- 現在時刻が営業時間外のとき、入店1時間後扱い（フォールバック）

### 端数処理
- **最終合計**: 税込金額を **百円単位切り上げ**（`Math.ceil(total/100)*100`）
- **TAX/SVC 表示額**: 切り上げ
- **セット半額 / 延長半額**: 切り捨て（`Math.floor`）
- **カンの半額**: 特殊価格 **700円**（通常半額750円 → 700円固定）

## 5. イベント・割引

### 割引イベント一覧と適用効果

| イベント | セット半額 | カン半額 | ショット半額 | シャンパン半額 |
|---|:-:|:-:|:-:|:-:|
| 女子会デー | × | ○ | ○ | ○（ブルー〜ゴールド）|
| お客様感謝DAY | ○（正規のみ）| ○ | ○ | × |
| セブンラック | × | ○ | ○ | ○（ブルー〜ゴールド）|
| 新規客（自動）| × | × | ○ | 1本目のみ半額 |
| R（自動）| × | × | × | 1本目のみ半額 |

### 排他 / 併用ルール

- **女子会デー ⇔ 感謝DAY**: 排他（同じ系統の割引のため併用不可）
- **セブンラック**: 女子会・感謝DAY の **どちらとも併用可**
- 併用時でも各種半額は **単一適用**（二重割引防止）
- **感謝DAY中はセット半額ボタンが無効化**（自動適用のため手動切替不可）

### セット半額（isSetHalfOff / appliedSetHalfOff）
- 正規のみ対象
- セット料金・延長料金の両方を **半額（切り捨て）**
- 感謝DAY中は自動ON・手動操作不可

## 8. 計算フロー

```
1. 客種 → セット料金・延長料金・指名料・T.C を決定
2. ゴールドカードがONなら セット¥0 / 延長¥1,000 に上書き
3. 正規かつ（セット半額 or 感謝DAY）なら セット・延長を half
4. 初回セット時間: R系・ゴールドカード=2時間、他=1時間
5. 延長本数 = 滞在時間 - 初回セット時間
6. 課税対象小計 = セット+延長 + 指名+同伴 + T.C + 複数指名料
                 + オーダー合計
7. TAX/SVC = 課税対象小計 × 35%（切り上げ）
8. 合計 = 小計 + TAX/SVC → 百円単位切り上げ
```

### 料金スケジュール表
- 入店時刻から営業終了までの各時間帯について、1セット目〜最終セットの合計金額を生成
- LOキャップ（24:38）を超えた場合、延長はその時刻までの本数で固定
