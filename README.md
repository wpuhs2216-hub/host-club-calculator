# Host Club Calculator (伝票計算システム)

ホストクラブ向けの伝票計算Webアプリケーションです。
複雑な料金体系（セット料金、延長、指名料、TAX/サービス料）を自動計算し、スムーズな会計業務をサポートします。

## 機能一覧

### 1. 基本計算機能
- **客層別料金設定**: 新規、R(内勤)、R(送り)、正規の4パターンに対応。
- **セット料金 & 延長**: 入店時間と現在時刻（または退店時間）に基づき、セット料金と延長料金を自動計算。
- **同伴料金**: 同伴オプションのON/OFF切り替え。
- **TAX/サービス料**: 小計に対してTAX(10%) + サービス料(30%) = 43%を自動加算（※計算式: `小計 * 1.43`）。

### 2. オーダー管理
- **商品リスト**: 缶モノ、ボトル、シャンパンなど、主要メニューをプリセット。
- **カスタム入力**: リストにない商品は「オリシャン / その他」から名称と金額を自由に入力可能。
- **数量変更・削除**: 追加したオーダーの数量変更や削除が簡単に行えます。

### 3. V4 新機能 (Advanced Features)
- **セット料金半額**: オプションでセット料金（延長含む）を半額に設定可能。
- **現在時刻連動**: 画面上部に現在時刻を表示し、リアルタイムで滞在時間と料金を計算。
- **シャンパン半額**: 特定のシャンパン（アスティ〜SPLゴールド）選択時、半額オプションが利用可能。適用時は商品名に「(半額)」が付与されます。
- **モバイル最適化**: スマホでの操作性を重視した縦長レイアウト。
